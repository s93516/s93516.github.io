# 個人作品集網站 (Personal Portfolio)

這是一個展示個人開發專案、影音多媒體與創作作品的響應式網站，同時作為本學期網頁設計課程的期末技術整合展現。

👉 **[點此瀏覽上線的完整網站 (GitHub Pages)](https://s93516.github.io)**

## 🌟 專案核心內容
本網站整合了前端網頁技術與多媒體視覺設計，包含以下四大展區：
* **🎮 獨立遊戲開發**：使用 Unity 2018 與 Fungus 開發的 18 世紀台灣傳統廟宇背景恐怖遊戲，結合 AIGC 生成之場景素材。
* **🎬 編劇與分鏡設計**：金穗獎格式短片劇本《停損》（Stop Loss），深入探討人性之間的矛盾與拉扯。
* **🖱️ 互動設計**：結合 UI/UX 概念與前端微動畫，打造流暢的使用者體驗。
* **🕶️ 沉浸式實境展演**：整合 AR 擴增實境互動影片與 360 度 VR 虛擬實境展場空間。

## 🛠️ 技術整合亮點 (課程技術應用)
本專案整合了本學期所學之核心網頁技術：
* **獨立樣式表 (CSS)**：使用獨立 `style.css` 進行模組化樣式管理。
* **響應式排版 (Flexbox)**：運用 `display: flex` 打造可自動適應螢幕大小的 2x2 網格卡片佈局。
* **互動特效 (Hover & Transition)**：實作卡片懸浮浮動、陰影加深等滑順過場動畫。
* **關鍵影格動畫 (Keyframes)**：設計網頁載入時的 `fade-in` 階層式浮現進場特效。
* **多頁面路由**：建立清晰的 HTML 檔案連結架構 (`<a>` 標籤跨頁跳轉)。

## 📂 檔案架構
* `index.html` - 網站首頁（整合四大專案入口）
* `style.css` - 全域樣式表（包含 Flexbox 與動畫設定）
* `game.html` - 遊戲開發與 AIGC 場景展示頁（建置中）
* `script.html` - 《停損》劇本與分鏡展示頁（建置中）
* `interactive.html` - 互動設計展示頁（建置中）
* `xr-exhibition.html` - AR / VR 實境展演頁面（建置中）
